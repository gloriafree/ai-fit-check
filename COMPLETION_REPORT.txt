================================================================================
CUSTOMIZATION GUIDE
================================================================================

Change Brand Color:
  → Edit Extensions.swift, change appPurple values

Change App Name:
  → Xcode: Target → General → Display Name

Change Server URL:
  → Edit APIService.swift, change @AppStorage default
  → Or: User changes in app Profile tab

Change App Group:
  → Find/replace all "group.com.aifitcheck.shared"
  → Update in both targets' capabilities

Change Bundle ID:
  → Xcode: Target → General → Bundle Identifier
  → Update Share Extension bundle ID
  → Update app group configuration
